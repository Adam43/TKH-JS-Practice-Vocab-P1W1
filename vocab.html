Vocab

  • The Internet

    The Internet, also known as the world wide web is a collection of servers that contains multitude of resources which can be served and displayed to clients using a web server. All these servers are computer servers that are in communication.

  • HTTP

    HTTP stands for Hypertext Transfer Protocol. It is a network protocal that allows you to access internet resources.

  • Client

    The client is what communicates with a server using the HTTP request to reciece information.

  • Server

    Server contains centralized resources that can be used to provide to the client PC. A collection of servers is what makes up a network of internet resources.

  • Web Browser

    Web Browsers allow you to interact with the resources that has been served from the server as well as be able to display websites.

  • Website

    Websites are a collection of webpages that displayed in the web browser.

  • Web Page

    Web pages are hypertext documents that are containted on the internet.

  • JavaScript

    JavaScript is also known as it's real name ECMASCRIPT. It is a web based scripting programming lanaguage that can be run on the brower as well be used as a server lanaugage called NodeJS.

  • HTML

    HTML is known as Hyper Text Markup Language. HTML is what is known as the bare bones of the web being able to show interactive and dynamic text.

  • CSS

    CSS is known as Cascading Style Sheets. CSS is a compliment to HTMl by being able to add styles, fonts, colors, into a web page and making your HTML resources even more attrative to look at.

  • URL

    URL stands for Universal Resource Link. URL are what is used for a webpage's address.

  • IDE / text editor

    IDE stand for integrated development environment. It uses a graphical user interface to display code as well as provide developer tools and software to build applications.

  • Command line / Terminal

    The command line is what a user uses to perform commands on a computer system. It allows a user to enter data and recieve output on execution.

  • BASH

    Bash stands for Bourne Again Shell. It is Unix shell and command language that can be used in the terminal to interact with your files and automate processes.

  • Operating system

    An operating system is categorized as a system software which gives you the ability to manage your digital resources.

  • Algorithm

    Algorithms are sequences processes and rules to be followed to perform a calculation for problem solving reasons. Algorithms are used by a computers

  • Frontend vs Backend Development

    The difference between frontend and backend development is that frontend web development is the user-interface whereas back-end means the application for the servers that cannot be seen by the users.

  • Programming vs Coding

    Coding is the process of translating and writing codes from one language to another, whereas Programming is the process of building an executable program that can be used to carry out proper machine level outputs.

  • Git vs Github

    Git is a local versioning software that enables developers to save their projects. And GitHub is a web-based platform that incorporates git’s version control features so they can be used collaboratively.

  • Github terms: push, pull, fork, clone, repository

    push: Refers to upload local repository content to a remote repository. pull: Allows developers to pull resources for collaboration reasons from local to remote. fork: Forking a repository allows you to test and debug with changes without affecting the original project. clone: It is used to make a copy of the target repository or clone it. repository: It contains the collection of the file as well as the history of changes made to those files.
